# NYC Restaurant Inspection Risk Analysis

## Project Overview
This project analyzes the risk factors associated with restaurant inspections in New York City. The goal is to understand how various factors contribute to the likelihood of issues during health inspections and to inform policy decisions that could improve public health.

## Dataset Information
The dataset used in this analysis consists of records from the NYC Department of Health and Mental Hygiene, covering various aspects of restaurant inspections from the last several years. It includes:
- Restaurant names and locations
- Inspection dates
- Violation codes and descriptions
- Grades assigned during inspections
- Any additional relevant metrics

## Key Findings
- **Correlation between Violations and Grades**: There is a significant correlation between the number of critical violations and the grades assigned during inspections.
- **Temporal Trends**: The data shows trends over time, suggesting that certain months see more violations than others, potentially due to seasonal factors.
- **Geographical Distribution**: Certain neighborhoods have higher violation rates, which could indicate a need for targeted inspection efforts.

## Project Structure
- `data/`: Contains raw and processed datasets used in the analysis.
- `notebooks/`: Jupyter notebooks that detail the analysis process and results.
- `src/`: Source code for data processing and visualization.
- `README.md`: Documentation for the project.
